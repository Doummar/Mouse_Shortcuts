# Mouse Answer Configuration

## General
- **toggle_binding** — Key or mouse button that toggles the addon on/off during review.
  Example (key):   `{"type": "key",   "key_int": 189,  "display": "½"}`
  Example (mouse): `{"type": "mouse", "button_int": 4, "display": "Mouse: Back Button"}`
- **addon_active**         — `true` / `false`  (toggled by the toggle binding)
- **show_answer_feedback** — `true` / `false`  (brief label after each rating)

## Action options
- `show_answer`  — Show the answer (question side only)
- `again`        — Rate Again (1)
- `hard`         — Rate Hard (2)
- `good`         — Rate Good (3)
- `easy`         — Rate Easy (4)
- `do_nothing`   — Do nothing

## Mouse button settings
- **right_click_question / answer**
- **right_double_click_question / answer**
- **middle_click_question / answer**
- **scroll_up_question / answer**
- **scroll_down_question / answer**
- **extra1_click_question / answer**  (back button)
- **extra2_click_question / answer**  (forward button)
